## 一、整体视觉定位

### 1.1 美术风格关键词
**古朴诙谐 · 水墨韵味 · Q版卡通 · 战场手札**

- **古朴**：借鉴明朝古籍版画、兵法阵图、古地图的视觉语言
- **诙谐**：角色造型夸张可爱，动作带有轻喜剧感，消解战争的沉重
- **水墨**：UI边框、特效、背景氛围融入水墨笔触
- **手札**：整体界面像一本摊开的抗倭将领手记

### 1.2 风格参考板

| 参考方向 | 参考来源 | 借鉴要点 |
|:---------|:---------|:---------|
| 角色造型 | 《王国保卫战》系列 | Q版比例、夸张表情、清晰辨识度 |
| 色彩质感 | 《江南百景图》 | 古朴色调、低饱和度、手绘感 |
| UI气质 | 《绘真·妙笔千山》 | 水墨勾边、卷轴式布局 |
| 战场氛围 | 明代《抗倭图卷》 | 阵型排布、旗帜、船只剪影 |

## 二、角色美术设计

### 2.1 角色造型总则
- **比例**：三头身Q版，头:身:腿 ≈ 1:1:1
- **线条**：手绘感描边，线宽2-3px，略有粗细变化
- **细节层级**：突出识别特征，简化琐碎装饰
- **动作风格**：攻击动作略带夸张，死亡有卡通化倒地动画

### 2.2 我方兵种详细设计

**藤牌兵**

| 设计要素 | 具体描述 |
|:---------|:---------|
| **整体印象** | 圆滚滚的憨厚壮汉，举着巨大的圆形藤牌，像移动的小堡垒 |
| **头部** | 戴戚家军红色笠帽（帽檐宽大），圆脸，表情坚毅中带憨厚 |
| **上身** | 身穿鸳鸯战袄（红色对襟短衣），外罩皮质护肩 |
| **下身** | 深蓝色束脚裤，草鞋 |
| **装备** | 左手持圆形藤牌（藤黄色，直径≈身高70%），右手持腰刀 |
| **待机动作** | 藤牌微微上下晃动，呼吸感 |
| **攻击动作** | 从藤牌后探出腰刀劈砍，随即缩回 |
| **受击动作** | 藤牌震动，身体后仰但站稳 |
| **死亡动作** | 藤牌倒下盖住身体，只露出一双脚 |

### 2.3 敌方角色详细设计

**设计总则**：倭寇造型需有辨识度，同时略带丑角化（歪嘴、独眼、猥琐表情），以消解暴力的不适感。

**浪人倭寇**

| 设计要素 | 具体描述 |
|:---------|:---------|
| **整体印象** | 衣衫褴褛的浪人，手持倭刀，表情凶恶但体型瘦小 |
| **头部** | 日式月代头（中间剃光），或乱发，表情歪嘴瞪眼 |
| **上身** | 破烂和服，袒露一肩，有简陋护甲片 |
| **下身** | 日式袴，草鞋或赤脚 |
| **装备** | 双手持倭刀（刀身微弯，比身高略短） |
| **攻击动作** | 双手高举劈砍，动作幅度大 |
| **死亡动作** | 倭刀脱手，身体转圈倒地，眼冒金星 |

**配色表**
- 和服：`#4A5A6A`（灰蓝，破旧感）
- 袴：`#5A4A3A`（灰褐）
- 倭刀：`#C0C0C0`（刀身）+ `#2A2A2A`（刀柄缠绳）

## 三、场景与战场设计

### 3.1 战场基础布局

**整体尺寸**：宽1920px，战场区域约1600×600px

**分层结构**（从远到近）：
1. **远景层**：水墨渲染的远山、海岸线、战船剪影
2. **战场底层**：沙土地面，有车辙、脚印纹理
3. **分界线**：战场中央的河道或木栅栏（半场分界）
4. **交互层**：士兵、敌人、特效
5. **前景层**：些许草丛、石块（装饰，不阻挡视线）

### 3.2 各关卡场景主题

| 关卡 | 战役 | 场景主题 | 特色元素 |
|:-----|:-----|:---------|:---------|
| 第1关 | 宁海之战 | 城郊田野 | 稻田、稻草人、远处城墙剪影 |
| 第2关 | 新河之战 | 河岸村落 | 木桥、河边芦苇、民居剪影 |

| 第4关 | 上峰岭之战 | 山地 | 松树、山石、雾气 |

### 3.3 核心场景元素——帅旗

| 设计要素 | 具体描述 |
|:---------|:---------|
| **位置** | 战场最左端，固定不可移动 |
| **造型** | 高大的帅旗杆（木质，有金属旗杆顶），悬挂红色大旗 |
| **旗面** | 红色底，金色"戚"字（或"戚家军"字样），边缘有飘带 |
| **待机动画** | 旗帜缓缓飘扬 |
| **扣血反馈** | 旗面破损、出现弹孔/刀痕；旗杆摇晃；屏幕红闪 |
| **濒死状态** | 旗面残破，旗杆倾斜 |
| **倒塌动画** | 旗杆折断，旗帜飘落，屏幕变灰（失败） |

## 六、战斗核心逻辑

### 6.1 士兵行为状态机

```
[生成] 
   ↓
[向右移动]（速度=兵种移速）
   ↓
[每帧检测] → 攻击范围内是否有敌人？
   ├─ 无 → 继续移动
   └─ 有 → [停止移动] → [攻击状态]
         ↓
         [攻击冷却计时]
         ↓
         [造成伤害] → 目标死亡？
               ├─ 是 → 返回[向右移动]
               └─ 否 → 目标仍在范围内？
                     ├─ 是 → 继续攻击
                     └─ 否 → 返回[向右移动]
```

### 6.2 碰撞与阻挡规则

| 情况 | 处理方式 |
|:-----|:---------|
| 我方单位相遇 | 可重叠通过，无碰撞 |
| 敌方单位相遇 | 可重叠通过，无碰撞 |
| 敌我单位相遇 | **触发碰撞**，双方停止移动 |
| 狼筅兵与敌方 | 敌方**强制停止**，无法穿越 |
| 藤牌兵阻挡 | 最多阻挡3个敌人，第4个及以后**可绕行**（如有空间） |
| 长枪倭寇穿刺 | 攻击可穿透我方前排，直接命中后排 |

### 6.3 胜负判定细则

**胜利条件**（满足任一）：
1. 所有波次敌人被清空，且场上无存活敌人
2. 我方任意单位抵达地图100%位置（敌方据点），并累计停留5秒

**失败条件**：
1. 戚继光帅旗生命值 ≤ 0

**帅旗护卫兵机制**：
- 攻击间隔：3秒
- 攻击力：30
- 攻击范围：全场
- 目标选择：优先攻击**距离帅旗最近**的敌方单位
- 子弹：必中，无视阻挡
